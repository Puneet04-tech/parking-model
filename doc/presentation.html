Machine Learning Pipeline

Model 1: Report Validator

  • LightGBM binary classifier
  • 55 enhanced features
  • Spatial-temporal data
  • Violation type indicators
  • User behavior patterns
  • Target encoding for reliability
  • Semi-supervised learning with pseudo-labeling

Accuracy: 80%+

ROC-AUC: 84%

Training: 165k+ samples

Model 2: Spatial Hotspot Detector

  • DBSCAN clustering algorithm
  • Identifies chronic violation zones
  • Groups nearby violations
  • Density-based clustering
  • Handles noise/outliers
  • Visual hotspot mapping

Algorithm: DBSCAN

Output: Hotspot clusters

Use: Spatial analysis

Model 3: Priority Predictor

  • LightGBM multi-class classifier
  • 11 ambient features
  • Location & time data
  • Area density metrics
  • Quick assessment mode
  • Real-time inference

Output: LOW/MEDIUM/HIGH

Use: Quick triage

4

Model 2: Spatial Hotspot Detection

DBSCAN clustering identifies chronic violation zones for targeted enforcement.

1

Density-Based Clustering

Groups violations that are spatially close together, identifying hot zones

2

Handles Noise

Automatically filters outliers and isolated violations as noise

3

No Predefined Clusters

Discovers natural cluster shapes without specifying number of clusters

4

Visual Mapping

Creates actionable hotspot maps for patrol route planning

Parameters

eps: 0.005 (neighborhood radius)
min_samples: 10 (minimum points for cluster)

Output

Cluster labels for each violation
Hotspot coordinates for enforcement
